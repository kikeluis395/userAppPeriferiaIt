# Users App Periferia IT- Ionic Angular

## Instalación

`npm install`

## Ejecutar

`ionic serve`

## Funcionalidades

- Lista de usuarios
- Búsqueda por nombre
- Pull to refresh
- Pantalla detalle
- Loader
- Manejo de errores
